TrapCalibration Data Product
========================================


FITS HDU Structure
------------------


+-------+------------------+----------+----------------------+
| Index | Name             | Datatype | Array Size           |
+=======+==================+==========+======================+
| 0     | Primary          | None     | 0                    |

| 1     | Image            | float64  | (20, 10)             |
+-------+------------------+----------+----------------------+
| 2     | ERR              | float64  | (1, 20, 10)          |

| 3     | DQ               | uint16   | (20, 10)             |

| 4     | TRAP_DENSITIES   | float64  | (4, 3)               |
+-------+------------------+----------+----------------------+
