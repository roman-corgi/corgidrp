FlatField Data Product
========================================


FITS HDU Structure
------------------


+-------+------------------+----------+----------------------+
| Index | Name             | Datatype | Array Size           |
+=======+==================+==========+======================+
| 0     | Primary          | None     | 0                    |

| 1     | Image            | float32  | (1024, 1024)         |

| 2     | ERR              | float32  | (1024, 1024)         |

| 3     | DQ               | uint16   | (1024, 1024)         |
