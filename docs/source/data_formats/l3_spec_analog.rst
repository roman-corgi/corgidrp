L3 Spectroscopy Data Product (Analog)
========================================


FITS HDU Structure
------------------


+-------+------------------+----------+----------------------+
| Index | Name             | Datatype | Array Size           |
+=======+==================+==========+======================+
| 0     | Primary          | None     | 0                    |
+-------+------------------+----------+----------------------+
| 1     | Image            | float64  | (125, 125)           |
+-------+------------------+----------+----------------------+
| 2     | ERR              | float64  | (1, 125, 125)        |
+-------+------------------+----------+----------------------+
| 3     | DQ               | uint16   | (125, 125)           |
+-------+------------------+----------+----------------------+
