L2b-Analog Data Product
========================================


FITS HDU Structure
------------------


+-------+------------------+----------+----------------------+
| Index | Name             | Datatype | Array Size           |
+=======+==================+==========+======================+
| 0     | Primary          | None     | 0                    |
+-------+------------------+----------+----------------------+
| 1     | Image            | float64  | (1024, 1024)         |
+-------+------------------+----------+----------------------+
| 2     | ERR              | float64  | (1, 1024, 1024)      |
+-------+------------------+----------+----------------------+
| 3     | DQ               | uint16   | (1024, 1024)         |
+-------+------------------+----------+----------------------+
| 4     | BIAS             | float32  | (1024,)              |
+-------+------------------+----------+----------------------+
