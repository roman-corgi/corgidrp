KGain Data Product
========================================


FITS HDU Structure
------------------


+-------+------------------+----------+----------------------+
| Index | Name             | Datatype | Array Size           |
+=======+==================+==========+======================+
| 0     | Primary          | None     | 0                    |
+-------+------------------+----------+----------------------+
| 1     | Image            | float64  | (1,)                 |
+-------+------------------+----------+----------------------+
| 2     | ERR              | float64  | (1,)                 |
+-------+------------------+----------+----------------------+
| 3     | PTC              | float64  | (400, 2)             |
+-------+------------------+----------+----------------------+
